Frexlet.Monoid.Involutive.Involution

We have an involution functor `.rev : Monoid -> Monoid` that sends a monoid `a : Monoid` to the monoid with the same carrier, but with reversed multiplication: a.rev.Sem Prod [x,y] = a.Sem Prod [y,x] We can use this functor to characterise involutions as self-inverse homomorphisms, and this characterisation allows us to treat involutive monoids quite abstractly.
